UserStory1: Login 

When user first opens application, they see an instagram login on the right and images that are examples of the dashboard once you are logged in on the left.Unlogged in user also sees an "About Us" tab that gives a brief description of what the web application does.The user is prompted to enter their instagram login information. There is also a link to sign up for Instagram if they don't have an account. If a user enters an invalid password or username, an error message will pop up. Otherwise, if it is their first login, they will be prompted with questions about their preferences and interests. If they have logged in before, they will be directed to their dashboard.
